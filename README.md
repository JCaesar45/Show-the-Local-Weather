# Show the Local Weather

## Overview
This project is a weather application that displays the current weather for the user's location, styled to resemble a local news channel broadcast. It dynamically updates the background and icons based on the weather conditions and provides a toggle button to switch between Fahrenheit and Celsius units.

---

## Features
- **Detects user location** via HTML5 Geolocation.
- **Fetches current weather data** using the freeCodeCamp Weather API proxy.
- **Displays weather information**: location name, weather icon, and temperature.
- **Dynamic backgrounds and icons** based on weather conditions (sunny, cloudy, rainy, snowy).
- **Toggle button** to switch temperature display between °F and °C.
- Styled to resemble a professional news broadcast with a sleek, modern interface.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- FreeCodeCamp Weather API proxy
- Geolocation API

---

## How to Use
1. **Open this project in CodePen** or your preferred IDE.
2. Ensure your browser supports **HTTPS** (required for Geolocation API).
3. Upon loading, the app will automatically request permission to access your location.
4. The current weather for your location will be displayed with a themed background.
5. Use the **"Show in °F" / "Show in °C"** button to switch temperature units.

---

## Customization
- Replace background image URLs in CSS to customize weather backgrounds.
- Use different icons or SVGs for a more polished look.
- Adjust styling to match your personal or branding preferences.

---

## Troubleshooting
- Make sure you allow location access in your browser.
- If geolocation fails, the app will prompt an error message.
- Ensure you’re serving the project via HTTPS to avoid geolocation issues.
